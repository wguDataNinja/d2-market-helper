# Developer Guide: Requirements & Implementation Steps

## 🎯 Requirements

1. **Separate FMV Logic**

   * All heuristics in `fair_price.py` so it can evolve independently.
2. **Graph-Based Normalization**

   * Treat each `ItemA:ItemB` ratio as an edge; source/dest weights.
3. **Ist as Base Unit**

   * Seed `Ist Rune` price = 1.0; derive all others.
4. **Confidence & Recency Propagation**

   * `confidence_flag` = True if any hop was low\_confidence.
   * `last_updated` = *oldest* timestamp along the conversion path.
5. **Output Format**

   * CSV with columns: `item_name`, `item_id`, `ist_value`, `confidence_flag`, `last_updated`.
6. **Configurable CLI**

   * Paths for input, item IDs, output.

## 🚀 Implementation Steps

1. **Argument Parsing**

   * Use `argparse` to accept `--input`, `--item-ids`, `--output`.

2. **Load Data**

   ```python
   with open(args.input) as f:
       trades = json.load(f)
   with open(args.item_ids) as f:
       item_map = json.load(f)
   ```

3. **Build Conversion Graph**

   ```python
   import networkx as nx
   G = nx.DiGraph()
   for pair, data in trades.items():
       a, b = pair.split(":")
       ratio = data["metadata"]["numeric_ratios"][ data["metadata"]["recommended_ratio"] ]
       G.add_edge(a, b, weight=ratio, conf=data["metadata"]["low_confidence"], ts=data["metadata"]["last_updated"])
       G.add_edge(b, a, weight=1/ratio, conf=data["metadata"]["low_confidence"], ts=data["metadata"]["last_updated"])
   ```

4. **Traverse from Ist**

   * BFS or Dijkstra (since weights multiplicative, BFS suffices if no negative).
   * Track for each node:

     * `ist_value` = product of weights along path
     * `conf_flags` = any edge.conf
     * `timestamps` = \[edge.ts ...]

5. **Propagate Metadata**

   * For each node, set:

     ```python
     confidence_flag = any(path_confs)
     last_updated = min(path_timestamps)
     ```

6. **Lookup Item IDs**

   * Flatten `item_ids.json` into `name → id` map.

7. **Write CSV**

   ```python
   import csv
   with open(args.output, 'w', newline='') as f:
   ```
